ID/CLASS 차이점?

공통점

id 선택자와 class 선택자는 특정 요소를 대상으로 스타일을 적용하기 위한 것입니다. 예를 들어 id와 class를 사용하면 모든 <p>요소가 아닌 특정한 <p>요소에만 스타일을 적용할 수 있습니다

ID

  • 요약 : '유일'한 요소에 스타일을 적용
  • id 선택자는 문서 안에 있는 단 하나의 요소에 스타일을 적용하는 경우에 사용합니다
  • 선언자 : #
  • 기본형 : #id명{} | 예시: #main { display: flex; }

CLASS

  • 요약 :'복수'의 요소에 스타일을 적용
  • class 선택자는 문서 안 복수의 요소에 스타일을 적용하는 경우에 사용합니다.
  • 선언자 : .
  • 기본형 : .class명{} | 예시:
    .script { padding: 20px 0; }
    #main .aside { width: 400px; }
